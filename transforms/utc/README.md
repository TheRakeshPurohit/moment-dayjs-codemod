# utc


## Usage

```
npx moment-dayjs-codemod utc path/of/files/ or/some**/*glob.js

# or

yarn global add moment-dayjs-codemod
moment-dayjs-codemod utc path/of/files/ or/some**/*glob.js
```

## Local Usage
```
node ./bin/cli.js utc path/of/files/ or/some**/*glob.js
```
